      # GitHub Actions からアップロード

      actions/upload-artifact を使用すると、必ず zip になってアップロードされるため、自前で zip にすると二重になってしまう。

      リリースにアップロードするだけなら actions/upload-release-asset を使えば勝手に zip にはされないので、解決できる。

      # Codespaces

      Codespaces が実装されたら、このメモも開発環境を作る必要がなくなるかも。
